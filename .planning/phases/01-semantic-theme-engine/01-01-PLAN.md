  <action>
Create `crates/textual-rs/src/css/theme.rs`:

1. Add RGB-to-HSL and HSL-to-RGB conversion functions (pure math, no external crate needed):
   - `rgb_to_hsl(r: u8, g: u8, b: u8) -> (f64, f64, f64)` where H is 0-360, S and L are 0.0-1.0
   - `hsl_to_rgb(h: f64, s: f64, l: f64) -> (u8, u8, u8)`

2. Add `lighten_color(color: TcssColor, delta: f64) -> TcssColor` that:
   - Extracts RGB from TcssColor::Rgb (returns unchanged for other variants)
   - Converts to HSL, adds delta to L (clamped 0.0-1.0), converts back
   - Negative delta = darken

3. Define `Theme` struct:
   ```rust
   pub struct Theme {
       pub name: String,
       pub primary: (u8, u8, u8),
       pub secondary: (u8, u8, u8),
       pub accent: (u8, u8, u8),
       pub surface: (u8, u8, u8),
       pub panel: (u8, u8, u8),
       pub background: (u8, u8, u8),
       pub foreground: (u8, u8, u8),
       pub success: (u8, u8, u8),
       pub warning: (u8, u8, u8),
       pub error: (u8, u8, u8),
       pub dark: bool,
       pub luminosity_spread: f64,
       pub variables: HashMap<String, TcssColor>,
   }
   ```

4. Implement `Theme::resolve(&self, name: &str) -> Option<TcssColor>`:
   - First check `self.variables` for an exact match on `name`
   - Parse the name: split on `-lighten-` or `-darken-` to extract (base_name, shade_direction, shade_number)
   - Look up base_name in the struct fields (match "primary" => self.primary, etc.)
   - If no shade suffix, return `TcssColor::Rgb(r, g, b)` directly
   - If shade suffix, compute: delta = shade_number * (luminosity_spread / 2.0), negate for darken
   - Apply lighten_color and return result
   - Return None for unknown names

5. Implement `pub fn default_dark_theme() -> Theme` with Python Textual's textual-dark values:
   - primary: (1, 120, 212) -- #0178D4
   - secondary: (0, 69, 120) -- #004578
   - accent: (255, 166, 43) -- #ffa62b
   - warning: (255, 166, 43) -- #ffa62b
   - error: (186, 60, 91) -- #ba3c5b
   - success: (78, 191, 113) -- #4EBF71
   - foreground: (224, 224, 224) -- #e0e0e0
   - background: (18, 18, 18) -- #121212
   - surface: (30, 30, 30) -- #1e1e1e
   - panel: blend surface toward primary at 10% -- approximately (27, 39, 49)
   - luminosity_spread: 0.15

6. Update `crates/textual-rs/src/css/mod.rs`:
   - Add `pub mod theme;`
   - Add `pub use theme::{Theme, default_dark_theme};`
